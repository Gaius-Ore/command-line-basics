# Command Line Basics

These are the commands I used for this assignment and their outputs:

1. **mkdir**  
   - Used to create the `projects` folder and the subfolders `week1` and `week2`.  
   - **Output:** New directories were created.

2. **cd**  
   - Used to move from the home directory into `projects`, and to switch between `week1` and `week2`.  
   - **Output:** Current working directory changed.

3. **touch**  
   - Used to create an empty file named `hello.txt` inside `week1`.  
   - **Output:** File `hello.txt` created.

4. **cp**  
   - Used to copy `hello.txt` from `week1` into `week2` and rename it as `hello_copy.txt`.  
   - **Output:** File `hello_copy.txt` created inside `week2`.

5. **rm**  
   - Used to delete the original `hello.txt` from `week1`.  
   - **Output:** File `hello.txt` removed.

6. **vim**  
   - Used to create and edit a file named `about_me.txt`.  
   - **Output:** A text file containing a short paragraph about my motivation for learning Node.js was saved.
